Bisyntax builder

Bisyntax builder

2021-2023

Bisyntax builder 是一個創新的語言學習工具，專注於幫助學習者理解和掌握雙語句法結構。這個項目旨在通過互動式的方法，促進學習者的語言發展和理解能力。

Bisyntax builder is an innovative language learning tool that focuses on helping learners understand and master bilingual syntactic structures. This project aims to promote learners' language development and comprehension abilities through interactive methods.

項目詳情

Project Details

  • 目標：開發一種創新的方法來幫助學習者理解和比較不同語言的句法結構
  • Objective:Develop an innovative approach to help learners understand and compare syntactic structures across different languages
  • 方法：結合視覺化工具和互動式練習，創建一個全面的學習平台
  • Method:Combine visualization tools and interactive exercises to create a comprehensive learning platform
  • 成果：提高學習者的語言學習效率，增強語法結構的理解和應用能力
  • Outcomes:Improve learners' language learning efficiency and enhance understanding and application of grammatical structures

Sunflower Reading Programme

Sunflower Reading Program

2020-2022

向日葵繪本閱讀計劃是專為兒童設計的語言學習項目，通過精心挑選的繪本故事來激發閱讀興趣，培養語言能力和創造力。該計劃特別注重情感和認知發展的結合。

The Sunflower Reading Program is a language learning project specifically designed for children, using carefully selected picture book stories to stimulate reading interest and develop language abilities and creativity. This program particularly emphasizes the combination of emotional and cognitive development.

項目詳情

Project Details

  • 目標：培養兒童的閱讀興趣和能力，促進語言發展
  • Objective:Cultivate children's reading interest and abilities, promote language development
  • 方法：使用多元化的繪本故事，結合互動活動和引導式閱讀
  • Method:Use diverse picture book stories, combined with interactive activities and guided reading
  • 成果：參與計劃的兒童表現出更高的閱讀興趣和更好的語言表達能力
  • Outcomes:Children participating in the program showed higher reading interest and better language expression abilities

Reading step by step

Reading step by step

2019-2021

Reading step by step 是一個循序漸進的閱讀能力發展項目，專注於分階段提升學習者的閱讀理解和分析能力。該項目特別關注有閱讀困難的學習者，提供個性化的學習路徑。

Reading step by step is a progressive reading ability development project that focuses on gradually improving learners' reading comprehension and analytical abilities in stages. This project particularly focuses on learners with reading difficulties, providing personalized learning paths.

項目詳情

Project Details

  • 目標：為不同水平的學習者提供系統化的閱讀能力發展路徑
  • Objective:Provide systematic reading ability development pathways for learners at different levels
  • 方法：通過階段性評估和量身定制的練習，逐步提升閱讀技能
  • Method:Gradually improve reading skills through staged assessments and customized exercises
  • 成果：學習者能夠有目標地提升閱讀能力，克服閱讀障礙
  • Outcomes:Learners are able to improve their reading abilities in a targeted manner and overcome reading difficulties
